# 利用ガイド

先生・生徒向けの詳しい利用情報です。

## 動作条件とプライバシー

センサーとマイクは、安全な接続であるHTTPSから開いたときに動作します。
GitHub PagesのURLはHTTPSなので、そのまま利用できます。ファイルを保存して`file://`で開くと計測できません。

ページを開くときはインターネット接続が必要ですが、アプリのコードには計測データをサーバーへ送る処理がありません。
GPSで使われるブラウザ・OS側の位置情報サービスについては、端末や学校の管理設定に従います。

## 授業での使い方（例：重力加速度 約9.8を読む）

「加速度」アプリで、落下中の加速度が**約9.8 m/s²**になることを生徒自身に読ませる流れです。

> ⚠️ **安全上の注意：端末を投げないでください。** 教師の管理下で、保護ケースを付けた端末を低い位置から
> 厚いクッションなどの柔らかい受け面へ落とします。学校・自治体の端末利用規則で落下実験が禁止されている場合は実施しません。

1. **加速度**を開き、`絶対値`タブを選ぶ
2. `▶`を押して、上記の安全条件で短い自由落下を記録する
3. `⏸`を押して一時停止する
4. グラフを**タップして拡大**する
5. **ピンチ**で、落下区間が見やすい幅まで広げる
6. **ドラッグ**で、区間を見やすい位置へ寄せる
7. 点を**タップ**して、約9.8 m/s²を読む

設計では高機能さよりも、授業が止まらないことを優先しています。最初の成功までが短く、
つまずいても生徒だけで回復でき、どのページも似た操作で使えることを目指しています。

## 計測方法の比較

| 比較軸 | このアプリ | PhET | phyphox | Vernier / PASCO |
|---|---|---|---|---|
| 主な体験 | **端末センサーで実測** | シミュレーション | 端末センサーで実測 | 主に各社センサーで実測 |
| 始め方 | **URL / QRコード** | Webまたはアプリ | アプリ | Webまたはアプリ |
| 追加機器 | **基本不要** | 不要 | 基本不要 | 実験に応じて各社センサー |

比較内容は各ツールの公式情報
（[PhET](https://phet.colorado.edu/)、[phyphox](https://phyphox.org/)、
[Vernier Graphical Analysis](https://www.vernier.com/product/graphical-analysis/)、
[PASCO SPARKvue](https://www.pasco.com/products/software/sparkvue)）に基づきます。
