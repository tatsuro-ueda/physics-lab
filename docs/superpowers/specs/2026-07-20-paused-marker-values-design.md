# 点の値を一時停止中だけ表示する設計

日付: 2026-07-20
対象: 共通時系列グラフ

## 目的

計測中は動くグラフの観察へ集中できるようにし、線をタップした点の値は一時停止中だけ表示する。グラフ右上の現在値は変更しない。

## 採用方式

- 計測中に線をタップしても、点、ツールチップ、選択時刻の数値を表示しない。
- 計測中の線タップで自動停止はしない。計測とグラフ操作を継続する。
- 一時停止後は、従来どおり線をタップして点の値を読める。
- 拡大、ピンチ、ドラッグ、線から離れたタップによる通常表示への復帰は、計測状態にかかわらず維持する。
- 加速度、ジャイロ、磁気、位置の時系列グラフへ共通適用する。

## 表示と案内

- 拡大表示では「⏸で止めてから線をタップして値を読む」と、前提と操作を一文で案内する。
- 加速度の必須チュートリアルは、既存の一時停止手順を値読みより前に維持する。
- 位置の必須チュートリアルは、値読みの直前に一時停止手順を追加する。
- ジャイロと磁気のヒントは「止めると読みやすい」ではなく「止めてから値を読む」と明示する。

## 実装境界

- 値読み可否の判定は `src/lab-common.js` の共通グラフに置く。
- ページ側は現在の計測状態をコールバックで渡し、独自の線タップ処理を追加しない。
- `src/lab-common.js` は正本とし、`build.py` で各配布HTMLへ埋め込む。
- 現在値、軸目盛り、計測データ、グラフ描画は変更しない。

## 検証

- 計測中の線タップでマーカーと点の値が表示されない。
- 一時停止後の線タップでマーカーと点の値が表示される。
- 計測中でも拡大、ピンチ、ドラッグ、通常表示への復帰が使える。
- 全生成HTMLに共通判定が埋め込まれる。
- `python3 -m unittest discover -s tests -v` と `git diff --check` が成功する。

## 非対象

- グラフ右上の現在値を隠すこと
- 線タップで自動的に一時停止すること
- 点だけ表示して数値を隠すこと
